Our Projects

  • Project 1: Algorithm Visualizer

    A tool to visualize and understand complex algorithms in real-time.

    Features:

    • Store Popularity Ranking: This involves ranking stores based on customer views or revenue. Implemented using sorting algorithms like QuickSort or MergeSort for efficient data organization.
    • Wi-Fi Coverage Optimization: Ensures optimal Wi-Fi coverage throughout the mall using Prim's or Kruskal's algorithm to map and optimize zones.

  • Project 2: Data Analysis Dashboard

    An interactive platform for exploring and analyzing datasets effectively.

    Features:

    • Shortest Routes to the Mall: Determines the shortest path from a specific starting point to the city mall using Dijkstra's algorithm.
    • Mall Revenue Tracking: Tracks and ranks revenue generated by different malls using a Binary Search Tree (BST) and other data structures for efficient updates and queries.

  • Project 3: Collaborative Task Manager

    A web application to manage team tasks and enhance productivity.

    Features:

    • Emergency Evacuation Plan: Designs efficient evacuation routes using graph traversal techniques like BFS and DFS.
    • Optimal Placement of ATMs: Determines the best locations for ATMs in the mall using the Floyd-Warshall algorithm to analyze and optimize distances.
