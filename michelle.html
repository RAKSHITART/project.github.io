Introduction to Project

In the City Design Project, my group chose the Mall Management System and explored different business cases to apply the algorithms. I worked on the following:

1. Parking Lot Management System

Purpose: This system assigns parking slots to customers based on proximity to mall entrances, optimizing convenience.

Implementation Details:

  • Parking slots are sorted using Selection Sort, ranking them by distance.
  • Breadth-First Search (BFS) dynamically assigns the nearest available slot to visitors.
View Code

2. Customer Navigation System

Purpose: Guides customers along the shortest path between two points in the mall.

Implementation Details:

  • The mall is represented as a graph where nodes are locations (e.g., shops) and edges are paths.
  • Dijkstra's Algorithm calculates the shortest path between two points.
  • A Binary Search Tree (BST) stores shop locations for efficient lookups.
View Code

3. Searching for Products in a Mall Directory

Purpose: Enables customers to search for specific products or categories (e.g., "electronics," "fashion") in the mall directory.

Implementation Details:

  • Rabin-Karp Algorithm is used for efficient searching by comparing hash values.
  • Hash values are calculated for the search term and matched against substrings in the directory.
