Mall Management and Waste Management System

1. Introduction

This combined system integrates Mall Management and Waste Management to ensure optimized customer experience and efficient waste handling within mall premises. The mall system ranks stores and places them optimally, while the waste management system ranks zones by waste levels and optimizes waste collection routes.

2. Mall Management System

2.1 System Overview

  • Store Ranking: Stores are ranked based on customer reviews or positive reviews.
  • Optimal Store Placement: Dijkstra’s algorithm is used to determine the shortest paths, ensuring top-ranked stores are placed in high-traffic areas.

2.2 Key Concepts

Store Ranking

Stores are ranked using two methods:

  • Rank by Total Reviews: Total number of reviews for each store.
  • Rank by Positive Reviews: Number of positive reviews each store receives.

Dijkstra’s Algorithm for Optimal Placement

Dijkstra’s Algorithm calculates the shortest paths from the entrance to other stores, ensuring the most accessible locations for popular stores.

3. Waste Management System

3.1 System Overview

  • Zone Ranking: Quick Sort is used to rank zones by waste levels.
  • Route Optimization: DFS ensures efficient waste collection routes.
  • Chute Placement: Kruskal’s Algorithm identifies optimal chute locations to minimize travel distances.

3.2 Key Concepts

Zone Ranking Using Quick Sort

Quick Sort ranks zones based on waste generation levels.

Route Optimization Using DFS

DFS plans efficient collection routes, minimizing redundant travel.

Chute Placement Using Kruskal’s Algorithm

Kruskal’s Algorithm determines the minimum spanning tree for optimal chute locations.
